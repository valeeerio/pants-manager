# Gestionale Sartoria

Gestionale per laboratorio sartoriale artigianale — gestione clienti,
lavori, pagamenti e consegne.

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS + shadcn/ui
- Prisma ORM
- PostgreSQL
- NextAuth.js
- Vercel (hosting)

## Avvio in sviluppo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
